# Your Own Theater (YOT)

Your Own Theater (YOT) is a web-based application that allows users to manage their movie preferences effortlessly. Users can create personalized watch lists, mark favorites, rate movies, and much more while seamlessly integrating with The Movie Database (TMDB).

## Features
- **User Authentication**: Secure login via TMDB authentication.
- **Custom Watch Lists**: Add movies to your watch list for future viewing.
- **Favorites**: Mark and categorize your favorite movies.
- **Ratings**: Keep track of and rate movies you've watched.
- **Navigation Friendly**: Intuitive and responsive user interface.

## Tech Stack
- **Frontend**:
  - HTML
  - CSS
  - JavaScript

- **Backend**:
  - Integration with TMDB API

- **Deployment**:
  - Hosted on [Render](https://yourowntheater.onrender.com/).

## Usage
1. Create an account on [TMDB](https://www.themoviedb.org/signup).
2. Log in to YOT using your TMDB credentials.
3. Start managing your movies with watch lists, favorites, and ratings.
